# PDF Pro Editor

محرر PDF احترافي يعمل في المتصفح مباشرةً، مستوحى من Adobe Acrobat.

## المميزات

| الفئة | الأدوات |
|-------|---------|
| **عرض** | تكبير/تصغير، ملاءمة، تدوير، ملء الشاشة |
| **نصوص** | إضافة نص منسّق (خط، حجم، لون، عريض/مائل/تسطير) |
| **تمييز** | تظليل، تسطير، شطب |
| **رسم** | قلم حر، ممحاة |
| **أشكال** | مستطيل، دائرة، سهم، خط |
| **إدراج** | صور، أختام (موافق عليه، رفض، سري...) |
| **تحرير** | تراجع/إعادة، نسخ/قص/لصق، حذف، ترتيب الطبقات |
| **التصدير** | حفظ صفحة كـ PNG، تصدير الملف كاملاً بـ PDF |

## المكتبات المستخدمة

- **PDF.js 3** — عرض صفحات PDF على `<canvas>`
- **pdf-lib 1** — تضمين التعليقات وتصدير PDF
- **Fabric.js 5** — طبقة تفاعلية للرسم والتعليقات

## تشغيل المشروع

لا يحتاج إلى خادم — افتح `index.html` مباشرةً في المتصفح، أو استخدم Live Server في VS Code:

```
افتح index.html → انقر زر Go Live
```

## اختصارات لوحة المفاتيح

| المفتاح | الوظيفة |
|---------|---------|
| `V` | أداة التحديد |
| `T` | إضافة نص |
| `P` | قلم حر |
| `E` | ممحاة |
| `R` | مستطيل |
| `C` | دائرة |
| `A` | سهم |
| `L` | خط |
| `Del` | حذف المحدد |
| `Ctrl+Z / Y` | تراجع / إعادة |
| `Ctrl+C/V` | نسخ / لصق |
| `+/-` | تكبير / تصغير |
| `←/→` | الصفحة السابقة / التالية |
